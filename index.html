HTML & CSS Basics Summary

h2>HTML Summary

HTML (Hypertext Mark Language) isused to define our page content, structure and meaning. You dont use it for styling purposes. Use CSS for that instead!

  • HTML uses "elements" to describe (annotate) content.
  • HTML elements typically have an opening tag, and a closing tag.
  • You could also have void (empty) elements like images.
  • You can also configure elements with attributes.
  • Theres a long list of available elements but you'll gain expirience over time, no worries.

Learn more about avaliable HTML elements on the MDM HTML element refrence.

CSS Summary

CSS Cascading Style Sheets is used for styling your page content.

  • Styles are assigned via property-value pair.s
  • You can assign styles via the "style" attributes.
  • To avoid code duplication, you typically use global styles (e.g. via the style element) instead
  • Aletrnatively, you can work with external stylesheet files which you can "link" to
    • When working with CSS, concepts like "interitance", "specificity" and the "box model" should be understood

Larn more about all avaliable propetries and values on the MDM CSS property reference.
